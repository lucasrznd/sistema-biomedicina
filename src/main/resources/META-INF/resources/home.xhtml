SGS - Home

Esse sistema foi desenvolvido para fornecer uma solução eficiente no controle de ampolas, permitindo o cadastro, edição e exclusão de dados relacionados a pacientes e ampolas armazenadas.

Com uma interface amigável e intuitiva, você poderá gerenciar facilmente o status das ampolas, indicando se estão armazenadas ou se já foram utilizadas.

Estamos comprometidos em fornecer uma ferramenta confiável e eficaz para garantir o controle preciso das ampolas, colaborando assim com o trabalho essencial realizado pela equipe de biomedicina da Santa Casa.
